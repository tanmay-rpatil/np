# Compilation 

```bash
$ make
```

# Notes on code execution

In one terminal, execute the server code by:

## Server

```bash
$ ./s 4444
# format ./s [port no]
```

## Client 

```bash
$ ./c 127.0.0.1 4444
# format ./c [server ip] [server port no]

```
